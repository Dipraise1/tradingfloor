# Trading Floor

The Floor is a modern trading journal and signals app built with React Native (Expo).

## Features

- **Journal**: Track trades, win rate, and strategies.
- **Signals**: Real-time signals feed backed by Supabase.
- **Insights**: Weekly bias and news (Coming Soon).
- **Admin Dashboard**: Post signals and manage content directly from the app.

## Tech Stack

- **Framework**: Expo (React Native)
- **Styling**: NativeWind (Tailwind CSS)
- **Backend**: Supabase
- **Navigation**: Expo Router

## Setup

1. `npm install`
2. Create `.env` with Supabase keys.
3. `npx expo start`
